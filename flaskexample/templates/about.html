Welcome to CourseXray

"student insights revealed for your future course"


CourseXray is a tool to help find the right online data course for you. Enter any keywords into the search bar and CourseXray will mine both course descriptions and course reviews for Coursera, Udacity, EdX, and Udemy. This provides a directed and more encompassing search than would be obtained using course titles or subject tags alone.

But that's not all!

The Term Xray also allows you to analyze student sentiment for any topic that is important to you. Use the Xray to see a distribution in polarity for any sentence in the student reviews where your topic was mentioned.

Project Background

I created CourseXray during the Insight Data Science Program in 2017 on a timescale of 3 weeks. Course descriptions and reviews were collected with a combination of web scraping and API calls using python libraries. Analysis was performed with the nltk and sci-kit learn packages, and the web application was created using Flask. Dynamic visualizations for the application were created using D3.js. The website was deployed with AWS EC2.
